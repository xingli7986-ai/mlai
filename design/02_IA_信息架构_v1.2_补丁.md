## 四、信息架构变化

### 4.1 三个广场的明确分工

| 页面 | 内容 | 目的 | 谁能看 |
|---|---|---|---|
| 印花衣橱 /products | 已上架商品 | 购买/团购 | 所有人 |
| 灵感广场 /inspiration | 创作过程展示(图+prompt) | 激发灵感,导流到工作室或商品 | 所有人 |
| 我的设计工作室 /my-studio | 创作工具 + 我的作品 | 创作 + 分享 + 商业化 | 登录后 |

### 4.2 内容流向

```
我的设计工作室创作 → 选择三个出口之一
                ├── 下单定制 → 私人订单
                ├── 发起团购 → 申请上架审核 → 通过后进印花衣橱
                └── 分享 → AI 审核 → 进灵感广场
                
灵感广场作品 → 用户操作三选一
            ├── 基于这个改造 → /my-studio
            ├── 发起团购 → 通知作者 → 同意后上架 → 印花衣橱
            └── 解锁 prompt → 看到创作详情
```
