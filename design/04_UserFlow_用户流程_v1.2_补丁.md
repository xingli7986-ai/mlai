## 流程 A:消费者创作 + 分享 + 商业化

```
登录 → 顶栏点"我的设计工作室" → /my-studio

进入工作室,选 4 种创作方式之一:
├─ 基于已有款式 → 进入款式库选择 → 携带参数进入创作
├─ 从零设计 → 空白画布开始
├─ 基于个人需求 → 填写场景/性格/身材问卷 → AI 推荐花型
└─ 上传图片素材 → 上传图 → AI 基于图生成印花

创作过程(用 4 个 AI 工具:印花生成 / 四方连续 / 图案上身 / AI 试衣)
        ↓
作品保存到"我的作品"
        ↓
选择 3 个出口之一:
├─ 出口 1:单独下单定制
│    → /my-studio 内点"下单定制"
│    → 选尺码/面料/数量
│    → 填地址
│    → 支付宝支付
│    → 进入个人订单(走 v1.1 订单状态机)
│
├─ 出口 2:发起团购
│    → 点"发起团购"
│    → 提交"上架审核"申请
│    → 管理员审核(标 P1 必做)
│    → 通过 → 创建团购,进入印花衣橱
│    → 团购规则同 v1.1(30 件/72h)
│
└─ 出口 3:分享到灵感广场
     → 点"分享"
     → 选择 prompt 公开方式(免费/付费/不公开)
     → AI 自动审核
     ├─ 通过 → 立即出现在 /inspiration
     └─ 不通过 → 退回修改 + 提示原因
```

## 流程 B:游客/消费者从灵感广场流转

```
访问 /inspiration → 浏览作品(无需登录)

发现喜欢的作品 → 点击进入 /inspiration/[id]
       ↓
四种动作:

动作 1:点赞/收藏/评论
       → 未登录:弹登录框
       → 已登录:直接执行
       
动作 2:基于这个改造
       → 未登录:弹登录框
       → 已登录:跳 /my-studio?from=inspirationId
       → 工作室携带原作品参数,用户可继续修改创作
       
动作 3:发起团购请求
       → 未登录:弹登录框
       → 已登录:发送通知给作者
       → 作者同意 → 走"上架审核流程" → 通过后开团
       → 作者拒绝 → 通知请求者
       
动作 4:解锁创作详情(prompt + 参数)
       → 未登录:弹登录框
       → 已登录 + 作品免费公开:直接显示
       → 已登录 + 付费解锁:
            → 选择支付方式(现金 / 积分)
            → 支付成功 → 显示 prompt + 参数
            → 解锁记录入库
       → 已登录 + 不公开:显示"作者未公开此作品创作详情"
```

## 流程 C:用户升级为设计师

```
普通用户持续创作和分享
       ↓
系统每日扫描用户作品累计数据
       ↓
检测到达到任一阈值:
├─ 累计赞数 ≥ 500
├─ 累计收藏数 ≥ 200
└─ 累计被购/团购件数 ≥ 30
       ↓
创建 UpgradeInvitation 记录(30 天有效期)
       ↓
通知用户:站内消息 + 短信
       ↓
用户在 30 天内响应:
├─ 接受 → 跳 /studio/join?invitation=xxx
│         → 填写补充信息、签协议
│         → 提交审核
│         → 管理员审核(7 工作日)
│         → 通过 → 升级为认证设计师
│              → 历史作品保留 + 自动加金色描边
│              → 创作次数升级为 20/天
│              → 分润切换到 10-20% 阶梯
│         → 拒绝 → 通知用户原因 + 建议改进
│
├─ 拒绝 → 邀请关闭(可主动重新申请)
│
└─ 30 天未响应 → 邀请过期(可主动重新申请)
```

## 流程 D:上架审核流程(用户作品想成商品)

```
用户在 /my-studio 选某作品 → 点"申请上架"
       ↓
填写商品信息:
├─ 设定面料档位(基础/中端/高端)→ 系统自动算价格区间
├─ 选择销售方式(团购 / 单独定制 / 两者皆可)
└─ 提交审核
       ↓
进入管理员审核队列
       ↓
管理员审核(参考 PRD §7.1):
├─ 通过 → 创建商品记录
│         ├─ 团购:进入印花衣橱 + 启动 72h 倒计时
│         └─ 单独定制:进入印花衣橱 + 长期可定制
│         → 通知作者
│
└─ 拒绝 → 退回 + 说明原因 + 通知作者
```

## 异常处理新增

### prompt 解锁支付失败

- 支付未成功 → 不显示 prompt + 退款 + 重试

### 上架审核被拒

- 通知作者,允许修改后重新提交,无次数限制

### 自动审核误杀

- 用户可申诉
- 管理员人工复核

### 升级邀请响应超时

- 30 天后自动过期,作品历史保留
- 用户随时可主动通过 /studio/join 申请
