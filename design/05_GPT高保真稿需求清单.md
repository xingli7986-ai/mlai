# 给 GPT:MaxLuLu AI 新增页面高保真稿需求清单

请基于现有的《MaxLuLu AI Design System》(Playfair + Inter + 思源宋体 + 思源黑体,
配色 Warm Ivory + Champagne Gold + Deep Noir),
为以下 4 个新页面 + 2 处修改各出一份高保真 UI 设计稿。

每份设计稿必须包含:页面目标、用户路径、模块结构、布局尺寸(桌面 1440 / 移动 375)、
字号、颜色规范、间距规范、组件状态、响应式规则、开发注意事项。

---

## 1. 灵感广场 /inspiration

**页面目标**:展示创作过程(图 + prompt + 参数),激发设计灵感

**模块需求**:

- Hero 区:slogan + 数据统计(总作品数、活跃创作者数、本周新增)
- 筛选栏:工具类型筛选(印花生成/四方连续/图案上身/AI 试衣)+ 排序(推荐/最新/最热/价格)
- 作品瀑布流:卡片显示效果图、标题、创作者、点赞/收藏/解锁状态
  - 认证设计师作品:金色描边 1.5px (#C8A875) + 右上角金色"认证"角标
  - 普通用户作品:无特殊视觉
  - 卡片右下角显示 prompt 公开状态(免费/¥XX 解锁/不公开)
- 右侧筛选边栏:工具类型、价格区间(基于团购定价)、有 prompt 公开的作品
- 顶部 CTA:"成为创作者 → 进入工作室"按钮

**响应式**:桌面 4 列瀑布流,平板 3 列,移动 2 列

---

## 2. 灵感作品详情页 /inspiration/[id]

**页面目标**:大图展示 + 创作信息 + 三个 CTA

**模块需求**:

- 左侧大图区(60% 宽):主图 + 缩略图切换 + 全屏查看
- 右侧信息区(40% 宽):
  - 创作者信息(头像 + 名字 + 角标)
  - 作品标题 + 描述
  - 点赞/收藏/分享按钮
  - prompt + 参数(根据公开状态显示):
    - 免费公开:直接显示
    - 付费解锁:显示模糊预览 + "¥XX 解锁"按钮
    - 不公开:显示"作者未公开"
  - 三个 CTA 按钮(垂直排列):
    - "基于这个改造"(金色主按钮)
    - "发起团购"(次按钮)
    - "解锁详情"(若适用)
- 底部:相似作品推荐(横向滚动)
- 评论区(下方)

**响应式**:移动端图在上,信息在下,CTA 固定底部

---

## 3. 我的设计工作室 /my-studio(主页)

**页面目标**:消费者创作工作台主入口,展示 4 种创作方式 + 我的作品

**模块需求**:

- Hero 区:slogan "穿你想象的样子" + 当日剩余创作次数(5 次/天)
- 4 种创作方式卡片(2x2 网格,大卡片):
  - 基于已有款式 → 卡片图:款式库示意 + "看款式选印花"
  - 从零设计 → 卡片图:空画布 + "自由发挥创意"
  - 基于个人需求 → 卡片图:问卷示意 + "AI 帮你推荐"
  - 上传图片素材 → 卡片图:上传箭头 + "用照片做印花"
- "我的作品"区(下方):
  - 作品列表(网格)
  - 每个作品显示:缩略图、标题、状态(草稿/已分享/已上架/已售)、操作按钮
  - 状态标签用 Badge 区分
- 顶部右侧:升级提示(如果用户接近升级阈值,显示"距认证设计师还差 XX 件销售")

**布局**:工作室专用布局(无消费者顶栏,有自己的工作区顶栏 + 返回主站按钮)

---

## 4. 我的作品列表 /my-studio/works

**页面目标**:管理我的所有作品,执行三个出口动作

**模块需求**:

- 顶部:筛选 tab(全部/草稿/已分享/已上架/已售出)
- 网格列表:
  - 大卡片显示作品缩略图
  - 卡片底部:标题 + 创作时间 + 状态 Badge
  - 卡片悬停显示快捷操作:编辑/分享/上架/下单/删除
- 卡片详情(点击展开或进二级页):
  - 完整信息
  - 出口三选一:下单定制/发起团购/分享灵感广场
  - 数据指标(被赞数/被收藏数/被购数,如已分享)

## 通用视觉要求

- 全部沿用 Design System 的颜色、字体、组件规范
- 所有数字(计数、价格)使用等宽数字 + 衬线对齐
- 认证设计师视觉系统在四处必须一致:
  - 灵感广场卡片
  - 印花衣橱卡片
  - 详情页作者署名
  - 评论区头像
- 输出格式:每页一张桌面稿(1440×900) + 一张移动稿(375×800)
- 同时输出每页的"模块标注规范图"(标注尺寸、间距、字号)
