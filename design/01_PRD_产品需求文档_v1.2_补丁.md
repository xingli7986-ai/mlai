# MaxLuLu AI 产品需求文档 PRD v1.2 补丁

> 基于 v1.1 增量补充。本补丁与 v1.1 一起阅读。
> 版本:v1.2 | 日期:2026-05-02
> 主要变化:新增"我的设计工作室" + "灵感广场",分润改为阶梯式,新增用户升级路径

---

## 一、新增产品定位说明

MaxLuLu AI 从"AI 服装设计灵感画廊 × 参团采购 × 个人定制 × 设计师创业平台"
扩展为消费者也可以创作 + 分享 + 商业化的 UGC 平台。

**新核心循环**:消费者创作 → 分享灵感广场 → 获得反馈 → 商业化(被买/被团购)→ 升级为设计师

---

## 二、新页面定义

### 2.1 我的设计工作室 /my-studio

**面向**:消费者(认证设计师走 /studio,功能更全)
**目的**:让普通消费者也能创作专属服装设计

**4 种创作入口**:

1. **基于喜欢的款式做相似印花** —— 从印花衣橱/灵感广场点"基于这个改造"进入,带入原作品参数
2. **从零设计印花** —— 空白画布开始
3. **基于个人需求生成花型** —— 输入场景(职场/约会/休闲)、性格关键词、身材比例,AI 推荐花型
4. **上传自己的图片做素材** —— 上传图片作为印花生成的灵感源

**给消费者的 AI 工具**(中等版):

- 印花生成
- 四方连续
- 图案上身
- AI 试衣

**3 个出口**:

1. 单独下单定制(¥1599-2999)
2. 发起团购(30 件起)
3. 分享到灵感广场(可不消费,纯分享)

**创作次数**:5 次/天(沿用 PRD v1.1 §6.4)

### 2.2 灵感广场 /inspiration

**面向**:所有人(公开,游客也能浏览)
**目的**:展示创作过程,激发设计灵感,促进商业化

**展示内容**:

- 创作效果图(必显)
- 创作者信息(必显)
  - 认证设计师:卡片金色描边 + 右上角金色角标
  - 普通用户:无特殊视觉标识
- prompt + 参数:**创作者自选**
  - 选项 A:免费公开
  - 选项 B:付费/积分解锁查看
  - 选项 C:完全不公开(只展示效果图)

**排序**:推荐算法 + 用户可切换排序

- 默认:推荐(综合点赞、收藏、团购量、浏览量、时效衰减)
- 备选:最新、最热、价格升降序、AI 工具类型筛选

**消费者动作**:

- 点赞 / 收藏 / 评论
- 解锁 prompt(付费或用积分)
- "基于这个改造" → 跳 /my-studio 携带参数
- "发起团购" → 提议团购(作者收到通知,同意后才能开团)
- "我也要这个" → 催促作者上架

### 2.3 灵感作品详情页 /inspiration/[id]

**面向**:所有人
**展示内容**:大图、作品故事、创作者信息、prompt 解锁状态、参数详情(已解锁时)、相似作品推荐
**操作**:点赞、收藏、评论、解锁 prompt、基于这个改造、发起团购

## 三、保留页面定义更新

### 3.1 印花衣橱 /products

**变化**:从"展示设计师作品"改为"**仅展示已上架可购买商品**"

- 必须有库存或开放团购的作品才会出现在这里
- 用户作品必须升级为商品(走"上架审核流程")才进印花衣橱
- 灵感广场的作品不一定在印花衣橱(可能只是分享,未上架)

**视觉变化**:认证设计师的商品卡片加金色描边 + 角标

### 3.2 商品详情 /products/[id]

**变化**:作者署名根据角色显示

- 认证设计师:头像 + 名字 + 金色"认证设计师"角标
- 普通用户:头像 + 名字(无角标)

## 四、收入分润(替换 PRD v1.1 §8)

### 4.1 阶梯式分润比例

**普通用户作品**(累计销售件数计):

| 累计销售件数 | 创作者分润 |
|---|---|
| 0 - 100 件 | 5% |
| 101 - 500 件 | 7% |
| 501 - 1000 件 | 9% |
| 1001 件以上 | 10% |

**认证设计师作品**:

| 累计销售件数 | 设计师分润 |
|---|---|
| 0 - 100 件 | 10% |
| 101 - 500 件 | 13% |
| 501 - 1000 件 | 16% |
| 1001 件以上 | 20% |

### 4.2 分润规则

- 分润基数 = 实付金额(扣除优惠券、邀约直减后)
- 工厂生产成本 + 平台运营成本由平台承担,从其余毛利中支付分润
- 退款后分润回收
- 结算周期:确认收货后 7 天
- 提现门槛:满 ¥100 支付宝提现
- 阶梯阈值按"累计已售件数"计算,不重置

### 4.3 prompt 解锁收入分润

- 创作者自定义解锁价格(范围 ¥1 - ¥30 / 次)
- 平台抽成 30%,创作者拿 70%
- 积分解锁:用户消耗积分换取查看权,创作者按比例获得积分回报(具体规则待定)

## 五、版权(PRD v1.1 §8 补充)

- 用户用 AI 生成的图案 → **创作者 + 平台共同持有**
- 创作者保留商业化权利:可在灵感广场免费分享或付费解锁 prompt
- 创作者不得将同一图案在 MaxLuLu AI 之外的平台二次商业化(需在用户协议体现)
- 平台保留使用作品做品牌宣传的权利

## 六、用户升级路径(新增)

### 6.1 普通用户 → 认证设计师

**自动升级邀请触发条件**(满足任一):

- 累计作品赞数 ≥ 500
- 累计作品收藏数 ≥ 200
- 累计作品被购/团购件数 ≥ 30

**触发后**:

1. 系统发站内消息 + 短信邀请
2. 用户进入 /studio/join 走入驻流程(填补充信息、签协议)
3. 管理员审核(7 个工作日内)
4. 通过 → 升级为认证设计师,作品历史保留

**主动申请**:

- 用户也可不等邀请,直接走 /studio/join 主动申请

### 6.2 升级后变化

| 项目 | 普通用户 | 认证设计师 |
|---|---|---|
| 作品视觉标识 | 无 | 金色描边 + 角标 |
| 创作次数 | 5 次/天 | 20 次/天(同 Studio) |
| 分润比例 | 5%-10% | 10%-20% |
| AI 工具 | 4 个(中等版) | 16 个(全功能) |
| 入驻 Studio | 不可 | 可 |

## 七、内容审核

### 7.1 灵感广场作品发布审核

- **AI 自动审核**:发布瞬间检查违规图案、抄袭、低俗内容
- 通过 → 立即可见
- 不通过 → 退回修改

### 7.2 上架商品审核(进入印花衣橱)

- 用户作品想上架成商品(参与团购或开放定制)
- 必须先走管理员人工审核
- 审核通过后进入印花衣橱

### 7.3 违规处理

- 用户/管理员可举报作品
- 管理员可下架、警告创作者、严重者封号
